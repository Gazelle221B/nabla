## 数学的正しさの検証結果

数学的・数値計算的観点からコードおよび数式を精査した結果は以下の通り。

1. **残差計算式の妥当性**:
   `pythagoreanResidual` の定義式 `|OA|² + |OB|² - |AB|²` は、頂点 $O$ を直角とする直角三角形の残差（三平方の定理 $a^2 + b^2 = c^2$ からの逸脱度）として数学的に正しい。
2. **不変条件テストの数学的正当性**:
   - **直角三角形の残差ゼロ**: $O(tx,ty), A(tx+a,ty), B(tx,ty+b)$ の頂点定義は常に直交ベクトル $OA=[a,0], OB=[0,b]$ を生成するため、常に直角三角形となり、残差が浮動小数点誤差の範囲内でゼロになるテストは数学的に正しい。
   - **平行移動不変**: 同一ベクトル $[dx, dy]$ による平行移動が距離および残差に影響しない性質を正しく検証している。
   - **回転不変**: 任意の回転角度 `angle` に対しても距離が保存される性質を考慮し、回転後の三角形の残差がゼロになることを正しく検証している。
   - **スケーリング**: $k$ 倍のスケーリングに対し、2乗距離が $k^2$ 倍になるスケール変化と残差の保存を正しくテストしている。
   - **脚交換**: 加算の可換性および距離の対称性に基づき、`pythagoreanResidual(O, A, B) === pythagoreanResidual(O, B, A)` が完全に成立することを検証している。
3. **既知例の期待値の正当性**:
   - `3-4-5` 直角三角形の残差が `0`
   - `5-12-13` 直角三角形の残差が `0`
   - 共線三角形 $O(0,0), A(3,0), B(4,0)$ の残差が `9 + 16 - 1 = 24`
   これらすべての期待値が数学的に正しく、ハードコードされた定数で検証されている。
4. **テストの自己確認性の有無**:
   テストコードは「検証対象の式を用いて期待値を作成し、それを同じ式に戻す」ような自己確認的（Tautological）な記述になっておらず、独立した理論的期待値（`0` や `24` などの既知の定数）と比較しているため、極めて健全である。
5. **MDX数式の正当性**:
   `sample-math.mdx` 内のインライン数式 `$a^2 + b^2 = c^2$` およびディスプレイ数式 `$$c = \sqrt{a^2 + b^2}$$` は、数学的な表現として正確である。

**指摘された数学的誤り**: **なし**（極めて正確に実装されている）
