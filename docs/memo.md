[A] # キセリョフの算術学とは

キセリョフの算術学（ロシア語：Систематический курс арифметики、英語では一般に Kiselev's Arithmetic と呼ばれる）は、ロシアの数学教育者アンドレイ・ペトロヴィチ・キセリョフ（Андрей Петрович Киселёв、1852年〜1940年）が1884年に著した中等学校向けの体系的な算術教科書のことを指します。この教科書は「Систематический курс арифметики для средних учебных заведений（中等教育機関のための体系的算術コース）」という正式名称で刊行され、以後ロシア帝国、そしてソビエト連邦にわたって、実に半世紀以上もの間、数学教育の標準的な教科書として使われ続けたことで知られています。

著者キセリョフについて



崩壊後、ロシアの数学教育界の一部では、キセリョフの教科書が持つ論理的な厳密さや明快さを再評価する動きがあり、現代の教科書と比較して「なぜ当時の数学教育がしっかりしていたのか」を説明する事例としてしばしば言及されます。日本語圏でも、AIや高等数学の話題と絡めて、ロシアの数学者の基礎教育の質の高さの背景としてキセリョフの教科書が紹介されることがあり、近年ではその英訳版（Kiselev's Arithmetic: The Classic Russian Mathematics Textbook）も出版され、海外の数学教育関係者や愛好家の間で改めて注目を集めています。

要約すると、「キセリョフの算術学」とは特定の学術理論や数学分野の名称ではなく、19世紀末から20世紀にかけてロシア・ソビエトの学校教育を支えた、極めて影響力の大きい算術教科書そのものを指す言葉だとご理解いただければと思います。
