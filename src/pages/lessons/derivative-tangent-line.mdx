## 平易な説明: 平均の速さと、その瞬間の速さ

車で 2 時間走って 120 km 進んだとき、平均の速さは時速 60 km です。でも、実際には信号待ちで止まっている瞬間もあれば、高速道路で時速 100 km を出している瞬間もあります。「平均の速さ」は区間全体をならした値であって、「ある一瞬の速さ(スピードメーターが指す値)」とは違うものです。

関数のグラフでも同じことが起きます。ある関数 f(x) のグラフ上で、2 点 (a, f(a)) と (a + h, f(a + h)) を結ぶ直線を**割線**と呼びます。割線の傾きは

$$
\frac{f(a+h) - f(a)}{h}
$$

で、これは「x が a から a + h まで変わる間の**平均変化率**」です。一方、a という 1 点だけでの「瞬間の変化率」を知りたければ、2 点目 a + h を a にどんどん近づけていく——つまり h を 0 に近づけていく——しかありません。この極限として得られる値が**微分係数**であり、そのときの割線が近づいていく先の直線が**接線**です。

## 形式的な定義

関数 f(x) の x = a における微分係数 f'(a) は、次の極限で定義されます。

$$
f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}
$$

この極限が存在するとき、点 (a, f(a)) における**接線**は、傾き f'(a) を持ち、この点を通る直線として定まります。

$$
y = f'(a)(x - a) + f(a)
$$

接線は「割線の極限」です。h を 0 に近づけるにつれて、割線は接点のまわりで少しずつ向きを変えながら、最終的に接線という 1 本の直線に近づいていきます。

## よくある誤解: 「h = 0 を代入すればいい」

微分係数の定義を見て、「じゃあ最初から h = 0 を代入すればいいのでは?」と考えたくなるかもしれません。しかし、それでは

$$
\frac{f(a+0) - f(a)}{0} = \frac{0}{0}
$$

となり、0 で割ることになってしまい定義できません。微分係数は「h = 0 を代入した値」ではなく、「h を 0 に**近づけていったときに割線の傾きが近づく先**」という**極限**です。h がどれだけ小さくても 0 でない限り、割線の傾きは(一般には)微分係数とぴったり一致しません——近づいていくだけです。この「代入」と「極限」の違いを、下の実験で数値を動かしながら確かめてみましょう。
