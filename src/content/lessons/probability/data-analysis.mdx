たくさんの数値データをそのまま並べて眺めても、全体としてどんな傾向があるのかはつかみにくいものです。**少数の代表的な数値に要約する**ことや、**2つの量を散布図(点の集まり)に描いて関係を見る**ことは、そのための基本的な道具です。この単元では、そうした要約の道具——平均・分散・相関係数——が、データの散らばりや2つの量の関係をどう捉えているのかを、実際に点を動かしながら発見していきます。

## 散布図と「代表値」

たくさんの点からなる散布図を考えます。この点の集まり全体を1つの点で代表させるとしたら、どこに置くのがふさわしいでしょうか——各点の x 座標の代表値、y 座標の代表値をそれぞれ求め、その組を「代表点」とするのが自然な発想の1つです。また、点の散らばり具合(1点の周りに集まっているか、広く散らばっているか)や、x が大きい点ほど y も大きくなる傾向があるかどうか(2つの量の間の**直線的な関係の強さ**)も、散布図から読み取れる大切な情報です。

では、その代表点や関係の強さは、点が1つ動くだけでどれくらい変わるものでしょうか。具体的な計算式や数値例はまだ示しません——まずは下の実験で、実際に点を動かしながら自分で確かめてみましょう。

## まず予想してみよう

下の実験には6個の点からなる散布図があり、そのうち色の違う1個だけを動かせます。**操作する前に予想してください**——その1個の点だけを大きく(他の点から離れた位置へ)動かすと、点全体の代表点(平均点)と、2つの量の直線的な関係の強さ(相関係数)は、それぞれどうなるでしょうか?

予想を選んでから、可動点の x・y 座標を動かして、平均・分散・標準偏差・相関係数がどう変化するかを観察してみましょう。

## よくある誤解: 「外れ値があっても平均は代表値として常に適切」

平均は最もよく使われる代表値ですが、**「データに外れ値が混ざっていても、平均さえ見ればいつでも集団の典型的な姿がわかる」というのは誤りです。** 上の実験で確かめた通り、6個のうち5個の点がまったく動いていなくても、残り1個を大きく動かすだけで平均は大きく変わります。これは、平均が全データの合計を個数で割った値である以上、避けられない性質です——極端に大きい(または小さい)値が1つ混ざるだけで、その値に平均が強く引っ張られます。

だからといって「平均は使いものにならない」わけではありません。大切なのは、平均という1つの数値だけを見るのではなく、**散布図そのものを見て外れ値の有無を確認する**習慣を持つことです。上の実験で、点を大きく動かしたときに平均点がどれだけ動くかを実際に観察できたはずです。

## 形式的な定義

$n$ 個のデータ $x_1, x_2, \ldots, x_n$ の**平均**は

$$
\bar{x} = \frac{x_1 + x_2 + \cdots + x_n}{n}
$$

で定義されます。平均からの散らばり具合を表すのが**分散**で、各データと平均との差(偏差)の2乗の平均として

$$
s_x^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i - \bar{x})^2
$$

と定義されます。分散は2乗の単位(たとえば「点」の分散なら「点²」)を持つため扱いにくく、**標準偏差** $s_x = \sqrt{s_x^2}$ を使ってもとの単位に戻すこともよくあります。

たとえば $x_1, x_2, x_3 = 1, 2, 3$ という3個のデータなら、平均は $\bar{x} = (1+2+3)/3 = 2$、偏差は $-1, 0, 1$、その2乗の平均(分散)は $(1+0+1)/3 = 2/3$ です。

2つの量 $x, y$ の間の関係を調べるには、**共分散**

$$
s_{xy} = \frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})(y_i-\bar{y})
$$

と、それを標準偏差の積で割って単位に依存しない形にした**相関係数**

$$
r = \frac{s_{xy}}{s_x s_y}
$$

を使います。$r$ は常に $-1 \le r \le 1$ の範囲に収まり(コーシー・シュワルツの不等式による)、$r$ が $+1$ に近いほど点は右上がりの直線に近づき、$-1$ に近いほど右下がりの直線に近づきます。$r$ が $0$ に近いときは、直線的な関係がほとんど見られないことを意味します——ただし、これは「直線的な」関係が弱いという意味であって、放物線のような曲線的な関係が強くても $r$ は小さい値になりうることに注意してください($r$ はあくまで**直線的な**関係の強さだけを測る量です)。

具体例として $x_1,\ldots,x_5 = 1,2,3,4,5$、$y_1,\ldots,y_5 = 2,3,5,4,6$ を考えると、$\bar{x}=3$、$\bar{y}=4$、偏差の組は $x: -2,-1,0,1,2$、$y: -2,-1,1,0,2$ です。偏差の積は $4, 1, 0, 0, 4$ で、その平均(共分散)は $9/5 = 1.8$ です。両方の分散はどちらも偏差の2乗の平均 $(4+1+0+1+4)/5 = 2$ で一致するので、相関係数は $r = 1.8 / \sqrt{2 \times 2} = 1.8/2 = 0.9$ となり、強い正の相関があることがわかります。

なお、上の実験で確かめられるように、**すべての点の $x$ 座標(または $y$ 座標)が同じ値**になると、その変数の分散が $0$ になり、分母 $s_x s_y$ が $0$ になるため $r$ は $0/0$ で数学的に定義できなくなります(点が縦一直線、または横一直線に並んだ状態です)。このときは「相関係数は定義されない」というのが正しい答えであり、$0$ や他の値を無理に当てはめてはいけません。

## 転用問題: 平均・分散・相関係数を求める

次の4人分のデータについて、以下の問いに答えてみましょう。

| 生徒 | $x$ | $y$ |
|---|---|---|
| A | 3 | 10 |
| B | 5 | 20 |
| C | 5 | 20 |
| D | 7 | 30 |

1. $x$ の平均と分散を求めなさい。
2. $x$ と $y$ の共分散と相関係数を求めなさい。

<details>
<summary>答えと考え方</summary>

1. $\bar{x} = (3+5+5+7)/4 = 20/4 = 5$。偏差は $-2, 0, 0, 2$ で、2乗の平均(分散)は $(4+0+0+4)/4 = 2$。

2. $\bar{y} = (10+20+20+30)/4 = 80/4 = 20$。偏差は $-10, 0, 0, 10$。偏差の積 $x_i$偏差 $\times$ $y_i$偏差は $20, 0, 0, 20$ で、共分散は $(20+0+0+20)/4 = 10$。$y$ の分散は偏差の2乗の平均 $(100+0+0+100)/4 = 50$。相関係数は $r = 10/\sqrt{2 \times 50} = 10/\sqrt{100} = 10/10 = 1$。実際 $y_i = 5x_i - 5$(3→10, 5→20, 7→30)という完全な直線関係になっているため、$r=1$ ちょうどになるのは筋が通っています。

</details>

平均・分散は「1個の数値でデータ全体をどう要約するか」、相関係数は「2つの量がどれだけ直線的に関係しているか」を測る道具です。どちらも便利な要約ですが、外れ値1つで大きく動きうることを踏まえ、数値だけでなく散布図そのものを確認する習慣を持つことが大切です。
