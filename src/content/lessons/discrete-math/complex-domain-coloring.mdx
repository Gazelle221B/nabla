## 前の単元からの続き: 複素数の「繰り返し」から、複素数の「関数」へ

前の単元「[フラクタル](../fractal-mandelbrot/)」では、複素数 $z = x + iy$ を座標のペア $(x, y)$
として扱い、$z_{n+1} = z_n^2 + c$ という規則を**何度も繰り返した**とき軌道がどうなるかを
観察しました。この単元では発想を変えて、複素数を1回だけ関数に通す場合を考えます。

複素数を入力すると複素数が出力される関数、たとえば $f(z) = z^2$ を考えましょう。実数の関数
$y = f(x)$ なら、横軸に $x$、縦軸に $y$ を取った平面上の曲線(グラフ)としてすぐに描けます。
では $w = f(z)$ のように、**入力も出力も複素数**(それぞれ実部・虚部という2つの数を持つ)の
関数は、同じように「グラフ」に描けるでしょうか?

## まず予想してみよう

下の実験は、複素平面上の各点 $z$ を、ある複素関数 $f$ について $w = f(z)$ を計算し、その
結果を**何らかの規則で色付け**したものです(規則の中身はまだ明かしません)。**操作する前に
予想してください**——複素数を入れると複素数が出てくる関数 $f(z) = z^2$ のような関数は、
「グラフ」に描けると思いますか?

## よくある誤解: 「複素関数は4次元が必要だから、グラフに描けない」

複素数 $z = x + iy$ は実部 $x$・虚部 $y$ という2つの数を持ち、出力 $w = f(z) = u + iv$ も
実部 $u$・虚部 $v$ という2つの数を持ちます。素朴に考えると、「入力2次元+出力2次元=合計4
次元の情報を、3次元までしか描けない紙やスクリーンに描くことはできない」と考えたくなります。
**しかしこれは誤りです。**

上の実験で確かめた通り、出力 $w$ の2つの成分(絶対値・偏角、または同じ情報を持つ実部・虚部)を
**色**という1つのチャンネル(色相・明るさ)に載せることで、2次元の入力平面1枚の絵の中に、
出力の情報の要点を収めることができます。4次元のグラフを諦めて描かないのではなく、**色という
工夫で2次元に収める**発想の転換がこの単元の核心です。ただし、色から数値を正確に読み取ることは
できません(人の目でわずかな色の違いを正確な数値に変換するのは困難)。正確な値は、実験の観察表
(プローブの数値入力とその計算結果の表)が担います——色の絵は構造の**要点**(とりわけ後述する
「巻き数」)を見るためのものです。

## 形式的な定義

### ドメインカラーリングの規約

複素平面上の各点 $z$ を、$w = f(z)$ について次の規則で塗ります(この実験の実装がそのまま
採用している規約です)。

- **色相(hue) = 偏角 $\arg(w)$**。$\arg(w) = 0°$ を基準に、反時計回りに $360°$ で色相環を
  一周します。
- **明るさ(lightness) = 絶対値 $|w|$ の対数スケール**。$|w| = 1$ を中間の明るさとし、
  $|w| \to 0$(零点に近づく)ほど暗く(黒に近づき)、$|w| \to \infty$(極に近づく)ほど
  明るく(白に近づき)ます。

この規約のもとでは、**零点(f(z)=0 となる点)は必ず黒に近づき、極(分母が0になり f が
発散する点)は必ず白に近づきます**——色の明るさだけでも、その点が零点に近いか極に近いかが
一目でわかります。

### 零点・極のまわりの「色の渦」と巻き数

零点や極のごく近くを、小さな円を描くようにぐるっと一周してみましょう。円周上の各点で色相
(=偏角)がどう変わるか観察すると、次のことが起こります。

- **零点のまわりでは、色相がその零点の重複度の回数だけ一周します。** 例えば $f(z)=z^2$ の
  零点 $z=0$(重複度2、$z^2=0$ の重根)のまわりを一周すると、色相は**2周**します。
- **極のまわりでは、色相が逆回りに(極の位数の回数だけ)一周します。** 例えば $f(z)=1/z$ の
  極 $z=0$(位数1の単純極)のまわりを一周すると、色相は**逆向きに1周**します。

この「色相が何回巻き付くか」という整数を**巻き数(winding number)**と呼びます。実験の観察表
では、プローブを囲む小円上で偏角の変化を実際に数値積分した**数値巻き数**と、あらかじめ手計算で
確定させた零点・極の重複度からの**期待巻き数**(下の表)を独立に突き合わせ、一致することを
確認できます。

| プリセット | 零点・極 | 重複度/位数 | 巻き数 |
|---|---|---|---|
| $f(z)=z^2$ | 零点 $z=0$ | 重複度2 | $+2$ |
| $f(z)=z^3-1$ | 零点 $z=1,\ e^{i2\pi/3},\ e^{i4\pi/3}$(各1つずつ) | 重複度1 | $+1$(それぞれ) |
| $f(z)=1/z$ | 極 $z=0$ | 位数1 | $-1$ |
| $f(z)=(z-1)/(z+1)$ | 零点 $z=1$ / 極 $z=-1$ | いずれも1 | $+1$ / $-1$ |

なぜ重複度・位数と巻き数が一致するのか(**偏角の原理**、argument principle と呼ばれる複素
解析の定理)の証明はこの単元の範囲外とし、実験と観察表で「一致する」という事実そのものを
確かめることに留めます。

## 転用問題

1. $f(z) = z^2$ について、$z = i$ での値 $f(i)$ と、その偏角 $\arg(f(i))$ を求めてください。
2. $f(z) = z^3 - 1$ の零点($f(z)=0$ となる $z$)を3つとも解いて、実験のプリセットが示す位置と
   一致することを確かめてください。
3. $f(z) = 1/z$ の原点(唯一の極)まわりの巻き数が $-1$(零点とは逆回り)になる理由を、
   偏角の変化から直感的に説明してください。

<details>
<summary>答えと考え方</summary>

1. $z = i$ のとき $f(i) = i^2 = -1$ です。$-1$ は実部 $-1$・虚部 $0$ の点なので、偏角は
   $\arg(-1) = 180°$、絶対値は $|-1| = 1$ です(実験でプローブを $(\text{re}, \text{im}) =
   (0, 1)$ にすると、観察表で $f(z)=(-1, 0)$・$|f(z)|=1$・$\arg(f(z))=180$ と確認できます)。

2. $z^3 - 1 = 0$ すなわち $z^3 = 1$ を解きます。$1$ の3乗根は、絶対値1・偏角
   $0°, 120°, 240°$ の3点です。極形式で書くと $z = 1$、$z = \cos 120° + i\sin 120° =
   -\tfrac12 + i\tfrac{\sqrt3}{2}$、$z = \cos 240° + i\sin 240° = -\tfrac12 - i\tfrac{\sqrt3}{2}$
   の3つです。検算: $\left(-\tfrac12+i\tfrac{\sqrt3}{2}\right)^3$ は、絶対値1・偏角120°の数を
   3乗するので絶対値 $1^3=1$・偏角 $120°\times 3 = 360° \equiv 0°$ となり、$1$ に一致します
   (同様に $240°\times 3 = 720° \equiv 0°$ で2つ目の根も検算できます)。実験の cubeMinusOne
   プリセットでこれら3点をプローブすると、いずれも $f(z) \approx (0,0)$ になることが確認
   できます。

3. 原点を中心とする半径 $r$ の円周上を反時計回りに一周すると、$z$ の偏角 $\theta$ は
   $0°$ から $360°$ まで(1周分)増えていきます。ここで $f(z)=1/z$ の偏角は
   $\arg(1/z) = -\arg(z)$ という関係にあります(絶対値 $|1/z|=1/|z|$ は偏角に無関係な量で、
   $1/z$ は $z$ を実軸に関して鏡映し絶対値の2乗で割ったものと同じ偏角を持つため、$z$ の偏角の
   符号だけが反転します)。つまり $z$ の偏角が $0°\to 360°$ と**増える**あいだ、$f(z)$ の
   偏角は $0°\to -360°$ と**減り**続けます——ちょうど1周分、ただし逆回りです。これが
   巻き数が $-1$(零点の巻き数 $+1$ や $+2$ とは符号が逆)になる直感的な理由です。

</details>

複素関数 $w=f(z)$ が「色を使えば2次元の1枚の絵で構造の要点が見える」こと、そして零点・極の
まわりで色相が(重複度・位数の回数だけ、符号を変えて)渦を巻くこと——この2つを自分の言葉で
説明できれば、この単元の理解は十分です。
