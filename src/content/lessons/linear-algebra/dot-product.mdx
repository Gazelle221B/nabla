## 前の単元からの続き: 三角比を「2つのベクトルの関係」に使う

前の単元「三角比と単位円」では、角度 $\theta$ に対応する単位円上の点の座標が $(\cos\theta, \sin\theta)$ になることを見ました。ここでは、この $\cos\theta$ を使って、**2つのベクトルがどれくらい「同じ向きを向いているか」を表す1つの数**を作ります。この数を**内積**(ないせき)と呼びます。

ベクトル $\vec{a} = (a_x, a_y)$、$\vec{b} = (b_x, b_y)$ に対して、内積 $\vec{a}\cdot\vec{b}$ には2通りの求め方があります。1つは座標成分だけを使う計算、もう1つは大きさとなす角だけを使う計算です。同じ量を2通りの別々の式で計算しているのに、この2つが常に一致する——これがこの単元の中心的な発見です。

## まず予想してみよう

下の実験では、原点から伸びる2つのベクトル $\vec{a}$、$\vec{b}$ の向きを、それぞれ角度スライダーで動かせます(大きさは固定します)。**操作する前に予想してください**——2つのベクトルがちょうど直角(90°)になるとき、内積 $\vec{a}\cdot\vec{b}$ はどうなるでしょうか?

予想を選んでから、実際にベクトルの向きを変えて、観察表に出てくる2つの計算結果(成分計算と $|\vec{a}||\vec{b}|\cos\theta$)を見比べてみましょう。

## よくある誤解: 「内積もベクトル(向きを持つ量)」

ベクトル同士の演算というと、ベクトルの足し算・引き算やスカラー倍のように「結果もまたベクトル(向きを持つ量)になる」と考えたくなるかもしれません。名前に「積」と付いているため、なおさらそう思いやすいところです。

しかし実験で確かめた通り、内積の結果は成分計算・$|\vec{a}||\vec{b}|\cos\theta$ のどちらで求めても、観察表には**ただ1つの数(スカラー)**が並びます。$\vec{a}$、$\vec{b}$ をどの向きに回転させても、観察表の内積の値そのものが「向き」を持つことはありません——変化するのは数値の大小と符号だけです。内積は「ベクトル×ベクトル→ベクトル」の演算ではなく、「ベクトル×ベクトル→数」の演算だという点が、通常のベクトル演算との決定的な違いです。

## 形式的な定義

ベクトル $\vec{a} = (a_x, a_y)$、$\vec{b} = (b_x, b_y)$ の**内積**は、座標成分を使って次のように定義されます。

$$
\vec{a}\cdot\vec{b} = a_x b_x + a_y b_y
$$

実験の「成分計算」の行はこの式そのものです。

内積にはもう1つ、大きさとなす角を使った表し方があります。$\vec{a}$、$\vec{b}$ のなす角を $\theta$($0 \le \theta \le \pi$)とすると、

$$
\vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos\theta
$$

が成り立ちます。この2つの式が一致することは、余弦定理から導けます。$\vec{a} = \overrightarrow{OA}$、$\vec{b} = \overrightarrow{OB}$ とすると、$\overrightarrow{AB} = \vec{b}-\vec{a}$ の大きさについて余弦定理より

$$
|\vec{b}-\vec{a}|^2 = |\vec{a}|^2 + |\vec{b}|^2 - 2|\vec{a}||\vec{b}|\cos\theta
$$

が成り立ちます(この導出は O, A, B が三角形をつくる、つまり $\vec{a}, \vec{b}$ がどちらもゼロベクトルでなく平行でもない場合のものです。平行な場合は $\theta = 0°$ または $180°$ で $\cos\theta = \pm 1$ となり、成分計算と直接比べると同じ関係がそのまま成り立つことが確かめられます)。左辺を成分で展開すると $|\vec{b}-\vec{a}|^2 = (b_x-a_x)^2+(b_y-a_y)^2 = |\vec{a}|^2+|\vec{b}|^2 - 2(a_xb_x+a_yb_y)$ となるので、両辺を見比べると $a_xb_x+a_yb_y = |\vec{a}||\vec{b}|\cos\theta$ が得られます。つまり、成分計算と $|\vec{a}||\vec{b}|\cos\theta$ は、同じ内積という量を2つの異なる材料(座標成分/大きさとなす角)から計算しているにすぎず、一致するのは偶然ではありません。

この式から、$\theta = 90°$($\cos 90° = 0$)のとき、$|\vec{a}|$、$|\vec{b}|$ がどんな値であっても $\vec{a}\cdot\vec{b} = 0$ になることが分かります。逆に $\vec{a}$、$\vec{b}$ がどちらもゼロベクトルでなければ、$\vec{a}\cdot\vec{b}=0$ であることと $\theta=90°$(2つのベクトルが**垂直**である)ことは同値です。実験でベクトルの向きを直角に近づけると、この関係を確かめられます。

## 転用問題: 成分計算・垂直判定・なす角

1. $\vec{a} = (2, 3)$、$\vec{b} = (-1, 4)$ の内積 $\vec{a}\cdot\vec{b}$ を求めてみましょう。
2. $\vec{a} = (3, k)$、$\vec{b} = (2, -6)$ が垂直になるように、$k$ の値を求めてみましょう。
3. $\vec{a} = (1, 0)$、$\vec{b} = (1, 1)$ のなす角 $\theta$ を求めてみましょう。実験で $\vec{a}$ の向きを $0°$、$\vec{b}$ の向きを $45°$ に設定すると、観察表のなす角θの行で確かめられます。

<details>
<summary>答えと考え方</summary>

1. $\vec{a}\cdot\vec{b} = 2\times(-1) + 3\times 4 = -2+12 = 10$。
2. 垂直 $\iff$ 内積が0。$\vec{a}\cdot\vec{b} = 3\times2 + k\times(-6) = 6-6k$。これが0になるのは $k=1$ のとき。
3. $\cos\theta = \dfrac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|} = \dfrac{1\times1+0\times1}{1\times\sqrt{2}} = \dfrac{1}{\sqrt{2}}$ なので $\theta = 45°$($\pi/4$)。実験でこの2つの向きに設定すると、観察表のなす角θの行が $45$ になることで確かめられます。

</details>

内積に2つの独立な計算方法があり、それらが常に一致すること、そして内積がスカラー(ただ1つの数)であり垂直であることの判定に使えることが、自分の言葉で説明できれば、この単元の理解は十分です。
